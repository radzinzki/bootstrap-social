Bootstrap Social Buttons

This project combines two cool things, social media buttons and transparent buttons. Based on excellent work done by lipis, this project provides additionaion styles, transparent and half-transparent.

As a big fan of transparent buttons, I felt the need to add and make available, transparent buttons for social media. Feel free to fork/contribute/suggest changes.

How to use

1. Include Bootstrap and Font Awesome

If you haven't done that already, include the latest Bootstrap and Font Awesome in your project.

2. Include the CSS or LESS

You have two options for enabling the social buttons in your project: vanilla CSS or source LESS. For vanilla CSS, just include the bootstrap-social.css file into your project.

For LESS, copy the bootstrap-social.less into your existing Bootstrap directory and import it into bootstrap.less via @import "bootstrap-social.less";. Recompile when ready.

3. Add some buttons!

Start using the buttons as you've normally do with the Bootstrap buttons that have an icon by adding the relevant class. For example: 

<a class="btn btn-block btn-social btn-twitter">
  <i class="fa fa-twitter"></i> Sign in with Twitter
</a>

For transparent or half-transparent buttons, add an extra class (btn-[social-network]-transparent or btn-[social-network]-half-transparent) like this: 

<a class="btn btn-block btn-social btn-vk btn-vk-transparent">
  <i class="fa fa-twitter"></i> Sign in with Twitter
</a>

Or if you want to just the icon button use it like this: 

<a class="btn btn-social-icon btn-twitter">
  <i class="fa fa-twitter"></i>
</a>

For transparent icons, add the btn-[social-network]-transparent class instead of btn-[social-network]: 

<a class="btn btn-social-icon btn-twitter-transparent">
  <i class="fa fa-twitter"></i>
</a>
